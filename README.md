# Everywear ── 分享你的穿著，僅在彈指之間

_組名：MacLab_

_組員：林賢哲、呂尚霖、林佳瑩_

_Android版本：6.0_


## Introduction

Everywear將**天氣**、**穿著**、**社群**三種元素結合在一起。使用者可以瀏覽其他使用者在不同地點、天氣狀態下的衣著照片，也可分享自己的穿著，APP會自動在照片上加上使用者的所在地以及天氣狀態。

透過Everywear，使用者可以了解某個地區的穿著樣式，以及其對應的天氣狀態。可以用於**旅行前的查詢**或是**了解流行趨勢**等功能。
